# Кластеризация клиентов торгового центра

**Задача:** Разделить покупателей на группы по возрасту, доходу и тратам, чтобы маркетологи знали, кому что предлагать.

**Что сделал:**
- Проанализировал 200+ клиентов 
- K-Means нашёл 5 чётких групп (молодёжь-тратитли, пенсионеры-экономы и т.д.)
- SQL-запросы для insights
- Интерактивный дашборд в Yandex DataLens

**Стек:** Python (Pandas, Scikit-learn), SQL, Yandex DataLens
